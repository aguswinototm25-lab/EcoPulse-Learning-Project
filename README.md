# EcoPulse - Langkah Kecil untuk Bumi 🌍

EcoPulse adalah proyek **pembelajaran pertama** saya dalam membangun antarmuka web (UI) yang modern dan responsif. Proyek ini berfokus pada kampanye hemat energi dan pengenalan energi terbarukan melalui desain landing page yang informatif.

## 🚀 Tentang Proyek Ini
Proyek ini dibuat sebagai sarana belajar untuk memahami:
- Struktur HTML5 yang semantik.
- Penggunaan **Tailwind CSS** untuk desain yang cepat dan efisien.
- Implementasi desain responsif (mobile-first).
- Simulasi interaksi AI Chatbot sederhana menggunakan JavaScript.

## ✨ Fitur Utama
- **Landing Page Responsif**: Tampilan yang menyesuaikan di layar HP, tablet, maupun desktop.
- **AI Chat Mockup**: Simulasi tanya jawab seputar solusi energi surya dan efisiensi listrik.
- **Panduan Hemat Energi**: Bagian edukasi untuk langkah praktis sehari-hari (Cabut steker, lampu LED, dll).
- **Newsletter UI**: Desain formulir berlangganan untuk update isu lingkungan.

## 🛠️ Teknologi yang Digunakan
- **HTML5**: Struktur utama dokumen.
- **Tailwind CSS**: Styling dan layouting via CDN.
- **Google Fonts & Material Symbols**: Tipografi dan ikonografi.
- **Vanilla JavaScript**: Untuk logika toggle menu mobile dan simulasi chat.
